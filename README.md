# UNP_PROJECT_MEDIPLUS_DATA_ANALYSIS_USING_MySQL

I did this project in collaboration with UNP, where I developed a Hospital Management System using SQL to streamline and analyze hospital operations. The system organizes data related to patients, doctors, appointments, prescriptions, medicines, and billing in a well-structured database. It provides key insights such as total patients, appointments, revenue, and medicine inventory.

Using SQL techniques like JOINs, GROUP BY, and aggregate functions, the system enables efficient data analysis, including tracking patient–doctor interactions, monitoring appointment statuses, and identifying outstanding payments. I also implemented practical queries to extract insights such as completed appointments, revenue calculations, and medicine usage patterns.

Overall, the project demonstrates how structured data management and SQL-driven analysis can improve operational efficiency and support informed decision-making in healthcare systems.
